# go-pokedex
Boot.dev project - A CLI Pokedex:  REPL client for the PokeAPI written in Go

### Build
``` go build ```

### Run
``` ./go-pokedex ```

### Usage
- `catch <pokemon_name>`: Attempts to catch a given pokemon
- `inspect <pokemon_name>`: Prints stats for a given pokemon if it's been caught
- `pokedex`: Lists the names of all pokemon that have been caught
- `help`: List all commands for the Pokedex
- `exit`: Exit the Pokedex
- `map`: Print the next page of locations
- `mapb`: Print the previous page of locations
- `explore <area_name | id>`: Prints the names of all pokemon in a given area
